## 三、产品体系

### 基础软件层

| 产品 | 定位 | 核心指标 |
|---|---|---|
| AMI | 分布式低时延消息中间件 | 60ns时延，3500万msg/s，2500+节点 |
| AMQ | 分布式低时延消息队列 | 30μs时延，>100万msg/s，云原生K8s |
| AMDB | 高性能嵌入式内存数据库 | CRUD 0.2~0.8μs，无锁，嵌入式SDK |
| AMI Platform | 分布式高性能计算平台 | 集成AMI+AMQ+AMDB，流水线/并发/编排 |
| ArchforceEuler | 华锐欧拉操作系统 | 基于openEuler，鲲鹏最佳性能 |

### 核心业务平台层

| 产品 | 定位 | 核心指标 |
|---|---|---|
| ATP T1/T3/T5 | 机构交易平台 | T1<1μs, T3<5μs, T5<20μs, 110+券商 |
| ATP T7 | 零售交易平台 | <80μs, 160万TPS, 2000万客户 |
| ATP T9 | 全球交易平台 | 港美股/东南亚/欧洲, 7×24, 云原生 |
| ARC | 实时风控平台 | 30μs穿透200规则, 39家券商 |
| AMD D1~D9 | 高速行情平台 | FPGA解码130ns, 90+系统, 2000+下游 |
| ACP | 实时清算平台 | 15分钟清算1000万账户 |
| AIM M9 | 投资管理平台 | 服务7万亿+资产, 6000+产品 |
| ATA | 统一TA系统 | 2.5万亿+资产, 覆盖1/4银行理财客户 |
| ATH | 高速交易总线 | >10万笔/秒, 异构系统桥接 |
| ASA | 基金/理财营销 | 获客→转化→交易→服务一体化 |
| AOB | 互联网证券平台 | 全球市场终端, 多家中资银行券商上线 |
